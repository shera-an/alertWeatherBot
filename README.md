#  Weather Bot «Метеор»

Telegram-бот для получения прогноза погоды с регистрацией пользователей, геолокацией и хранением данных в PostgreSQL.

---
##  Ссылка на бота

**@WeatherAlertBot** — [перейти в Telegram](https://web.telegram.org/k/#@alertWeatherAsistant_bot)
##  Функциональность

- Регистрация пользователей (имя, город)
- Текущая погода по городу
- Почасовой прогноз на 24 часа
- Прогноз на 3 дня и на неделю
- Погода по геолокации (отправка местоположения)
- Изменение имени и города через настройки
- Красивые сообщения с эмодзи
- Указание источников данных (Open-Meteo, Яндекс.Карты)

---

##  Технологии

- **Java 17**
- **Telegram Bot API** (библиотека `telegrambots`)
- **PostgreSQL** + **Flyway** (миграции)
- **Open-Meteo API** (погода)
- **Яндекс Геокодер API** (координаты по городу)
- **SLF4J + Logback** (логирование)
- **Maven** (сборка)
- **Docker** (для локальной разработки)
